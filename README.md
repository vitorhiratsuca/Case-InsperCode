# Case---InsperCode
Repository for carrying out a group activity for the InsperCode organization's selection process.

Para quem precisar, ao executar pela primeira vez:

    python -m venv .venv

    source .venv/bin/activate # Mac e Linux #

    .venv\Scripts\Activate.ps1 # WINDOWS #

    pip install requirements.txt

SEMPRE QUE INICIAR RODE:

    source .venv/bin/activate # Mac e Linux #

    .venv\Scripts\Activate.ps1 # WINDOWS #


PARA RODAR O SERVER:

    python manage.py runserver
